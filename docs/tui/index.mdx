The TUI is a full-screen terminal application running at 30fps, built with [OpenTUI](https://opentui.com) and React. It renders the same domain data as the CLI but with persistent views, real-time updates, and an interactive chat interface.

## Keyboard Navigation

| Key      | Action           |
| -------- | ---------------- |
| `d`      | Dashboard screen |
| `q`      | Quests screen    |
| `s`      | Skills screen    |
| `c`      | Chat screen      |
| `r`      | Refresh data     |
| `Ctrl-C` | Quit             |
