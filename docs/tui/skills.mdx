---

Press `s` to open the skills screen.

Displays the full skill tree hierarchy. Each node shows:

- Skill name
- Current level
- XP progress bar toward next level
- Total XP accumulated

Parent skills are shown with their children indented beneath them. Skills with higher levels are visually distinguished.
