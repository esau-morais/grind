The Grind web app runs locally and gives you a browser-based view of your data alongside a full AI companion chat.

<CodeGroup>
  ```bash installed
  grindxp web start
  ```

```bash dev
bun web
```

</CodeGroup>

Then open [http://localhost:3000](http://localhost:3000).
